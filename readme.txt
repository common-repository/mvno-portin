== Installation ==
###Before You Start
Here are some things to know before you begin this process.
- You need to down a plugin called mvno-settings from the store before going to use it
- You should be signed up with https://mvnop.com/ (MVNOP)
- They will provide you with MVNO number and auth code
- Add that number and auth code into settings page
- Deleting removes the connection between MVNO Portin and Pareteum MVNO Services, and uninstalls the plugin from your site.
